## Creator

## Technical description

* Build year: 2006
* Builder: ALSTOM
* Number of cars: 5 (continuous train)
* Power supply: 1200V overhead wire
* Safety systems: Train Stop, ATP/ATO

## OpenBVE features

* Static 2D cab
* No exterior view
* Realistic sounds

## Installation
